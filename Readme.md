# PROBLEMÁTICA


- Los servicios actuales recomiendan música basada principalmente en género, artistas similares e interacción con la aplicación (búsqueda, escucha, saltar o guardar)
- Los usuarios se estancan en "burbujas musicales" sin descubrir artistas emergentes o artistas independientes
- Falta de personalización basada en características musicales específicas y estado de ánimo de la persona


# PROPUESTA DE VALOR


> "Descubre tu próxima canción favorita basada en lo que realmente escuchas, no en lo que millones escuchan"


- **Análisis de audio avanzado**: Recomendaciones basadas en características musicales reales
- **Correlación multidimensional**: Similaridad por features
- **Descubrimiento de artistas emergentes**: Rompiendo burbujas algorítmicas por ser más simple


# MODELO DE NEGOCIO


## 1. Modelo B2B: API para Plataformas Existentes


**Concepto**: Vender tu algoritmo de recomendación como servicio a otras empresas desarrollando una API que otras plataformas pueden integrar (Spotify, Deezer, Apple Music podrían usarlo para mejorar sus recomendaciones). La finalidad es lograr mayor cantidad de horas de uso de las aplicaciones existentes.


**Monetización**:
- Pago por uso: Cargo por cada consulta a la API
- Licencia mensual/anual: Según el volumen de usuarios de la plataforma cliente


**Ventajas**:
- Alto potencial de ingresos con pocos clientes
- No competimos directamente con aplicaciones ya establecidas que ya tienen contratos vigentes con las disqueras


## 2. Modelo B2C


**Concepto**: App gratuita con recomendaciones limitadas. Versión premium con recomendaciones ilimitadas.


**Monetización**:
- Pago por uso: Suscripción premium
- Comisiones por ventas: (Enlaces de afiliado a Spotify, Apple Music, Amazon Music). Dando mayor probabilidad de encontrar lo buscado
- Ticketing: Comisión por venta de entradas a conciertos de artistas recomendados
- Merchandising: Venta de productos de artistas descubiertos en la plataforma


**Ventajas**:
- Múltiples flujos de ingresos
- Se crea un ecosistema alrededor del descubrimiento musical
- Los usuarios pagan por valor tangible


## 3. Modelo B2B2C para Marcas y Artistas


**Concepto**: Servicios de recomendación personalizada para marcas que quieren conectar con audiencias específicas.


**Aplicaciones**:
- Sellos discográficos usan tus datos para lanzamientos estratégicos
- Agencias de marketing crean campañas basadas en datos recopilados


**Monetización**:
- Consultoría musical: Análisis de tendencias para sellos discográficos
- Campañas patrocinadas: Artistas emergentes pagan por mayor visibilidad
- Datos de mercado: Venta de datos sobre tendencias musicales


**Ventajas**:
- Margen muy alto en servicios de consultoría
- Posicionamiento como autoridad en el sector
- Menor dependencia del usuario final
